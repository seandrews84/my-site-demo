About Computer Science 2

Course Overview

Computer Science 2 is designed to build upon the foundation established in Computer Science 1, expanding students' programming knowledge and introducing more advanced concepts in computer science. This course emphasizes practical application, critical thinking, and collaborative problem-solving.

Learning Objectives

  • Master advanced object-oriented programming concepts
  • Design and implement complex data structures
  • Analyze algorithm efficiency and optimization techniques
  • Develop solutions to real-world computing problems
  • Work effectively in collaborative programming environments
  • Create well-documented, maintainable code

Prerequisites

Students should have successfully completed Computer Science 1 or have equivalent knowledge, including:

  • Fundamental programming concepts
  • Basic control structures (loops, conditionals)
  • Function implementation
  • Basic data types and variables
  • Elementary problem-solving approaches

Course Structure

  • Weeks 1-3

    Review & Advanced OOP

    Review of programming fundamentals, deep dive into object-oriented programming with inheritance, polymorphism, and encapsulation.

  • Weeks 4-6

    Data Structures

    Implementation and application of lists, stacks, queues, trees, and hash tables.

  • Weeks 7-9

    Algorithm Analysis

    Big-O notation, sorting and searching algorithms, and optimization techniques.

  • Weeks 10-12

    Advanced Topics

    Recursion, dynamic programming, and introduction to software design patterns.

  • Weeks 13-15

    Final Project

    Collaborative capstone project applying all course concepts to solve a real-world problem.
